This is a folder of my student works in AutoCAD and Autodesk Inventor. It contains projects focused on 2D technical documentation and 3D modeling.

In AutoCAD, I developed the TS1 project, which is technological documentation including earthworks, formwork, and crane design. Furthermore, the folder contains civil engineering drawings PSI 1 and PSI 2, a steel hall project, the DZUS road project, and the RBZS reinforcement drawing.

In Inventor, I created a 3D model of the CHESS set with an animation of the Carlsen vs. Nakamura match and a detailed replica of the Lavaforge Monolith hammer from the game.
